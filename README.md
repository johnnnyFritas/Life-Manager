# Life-Manager 🚀

Life-Manager is an Android application designed to help users optimize their daily routines and tasks. The project focuses on a clean architecture, modern UI patterns, and robust data management.

## 🛠 Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** Android Fragments & XML with **View Binding**
- **Architecture:** MVVM / Repository Pattern
- **Backend Integration:** Firebase Authentication and springboot with Java
- **Local Tools:** Git for version control

## ✨ Features (In Development)

- [x] **Modular Navigation:** Using a centralized `Navigator` object for fragment transactions.
- [x] **Clean Authentication Flow:** Repository-based logic for handling user sessions.
- [x] **Modern UI Implementation:** Clean XML designs with professional English annotations.
- [ ] **Task Management:** Create, edit, and track daily goals.
- [ ] **Data Persistence:** Integration with Spring Boot API and PostgreSQL.

## 📂 Project Structure

The project follows standard Android package conventions to ensure scalability:
- `ui`: Fragments and UI-related logic.
- `repository`: Data handling and abstraction layer.
- `utils`: Helper classes and navigation management.
- `databinding`: Safe view interactions via View Binding.
